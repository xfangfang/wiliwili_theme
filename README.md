# wiliwili 自定义布局

## 如何使用

1. 打开 wiliwili 配置目录（应用设置/实用工具 有快捷打开方式）
2. 将本项目的内容复制到 wiliwili 配置目录下 theme 文件夹内
3. 重启应用，在设置中选择自定义布局即可

复制好之后，配置目录应该长这样：

```
wiliwili
├── theme
│   ├── layout_1
│   │   ├── resources_meta.json
│   │   └── xml
│   ├── layout_2
│   │   ├── resources_meta.json
│   │   └── xml
│   └── layout_3
│       ├── resources_meta.json
│       └── xml
└── wiliwili_config.json
```
